Custom convertors for specific types, and supporting classes.

To activate a converter, the following entry has to appear in faces-config.xml:

  <converter>
    <converter-for-class>FqcnofTargetType</converter-for-class>
    <converter-class>be.peopleware.jsf_I.convert.ConverterClassName</converter-class>
  </converter>

More converters for value types can be found in ppw-value.
