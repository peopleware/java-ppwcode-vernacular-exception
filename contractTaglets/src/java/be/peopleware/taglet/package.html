The PeopleWare Taglet Module is a module that allows the user to create his own handlers for custom taglets in java source code.

The module contains one abstract generic class:

  • TagletRegistrar

The custom taglet is a javadoc-type taglet which can be used to customize Javadoc output.
(see http://java.sun.com/j2se/1.4.2/docs/tooldocs/javadoc/taglet/overview.html )
This module uses standard SUN's Javadoc doclet extension mechanism and encapsulates some common predefined behaviour in base class. By subclassing this base class custom taglets get most of the necessary behaviour.

Writing a Custom Taglet

  1. Extend class be.peopleware.taglet.TagletRegistrar
  2. Implement abstract methods from the base class.
  3. Implement the static method static void register(Map tagletMap).
    For example, the following implementation for the class Foo should do the trick:
    public static void register(Map tagletMap) {
    TagletRegistrar.registerTaglet(tagletMap, new Foo());
    }
  4. Compile your doclet.
    Use javac compiler version 1.4.0 (or later) in the Java 2 SDK.
    The required class files are in the lib\tools.jar file in the SDK.
  5. Run the javadoc tool using the -taglet and -tagletpath options.
