The PeopleWare Standalone Taglets currently consist of the following taglets:

  • IdeaTaglet for marking a tought that might we worth checking into later.
  • InvarTaglet for listing type invariants of a class/method/...
  • MudoTaglet for marking things that must be done for a future release.
  • PostTaglet for listing post conditions of a method.
  • PreTaglet for listing pre conditions of a method.
  • MudoTaglet for passing a question to a person via the source control management system.
  • ResultTaglet for listing the result that must be obtained by the method.
  • TodoTaglet for marking things that must be done when time is available for them.
